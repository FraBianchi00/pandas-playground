Repo di apprendimento per padroneggiare pandas attraverso dataset reali.
Parte del mio percorso di Fase 1 (Fondamenta) del roadmap Data Science.

## Setup

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
jupyter notebook
```
